CI/CD Pipeline Demo

A simple microservice demonstrating automated CI/CD with Jenkins, GitHub, and Ngrok.

Pipeline Overview

This microservice is automatically built, tested, and deployed using a Jenkins CI/CD pipeline triggered by GitHub push events via Ngrok.

  • GitHub Repository: Stores the source code.
  • Ngrok: Exposes the local Jenkins server to the internet.
  • Jenkins: Automates build, test, and deployment.
  • Cloud Platform: Hosts the deployed application.
